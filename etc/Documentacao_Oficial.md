# 📖 Documentação Oficial e Guias

Links para documentação oficial, guias e referências técnicas.

## Documentação Oficial

- [**Documentação Oficial Swift**](https://developer.apple.com/documentation/swift) - A fonte de verdade sobre a linguagem Swift, com guias, referências e tutoriais da própria Apple.
- [**Swift.org**](https://www.swift.org/) - O site oficial da comunidade Swift, com informações sobre o projeto open source, downloads e documentação completa.
- [**Swift Evolution**](https://github.com/apple/swift-evolution) - Repositório que rastreia propostas de mudanças e melhorias na linguagem Swift.

## Guias e Recursos Compreensos

- [**Hacking with Swift**](https://www.hackingwithswift.com/) - Vasta coleção de tutoriais, projetos e cursos em Swift, ideal para iniciantes e desenvolvedores experientes.
- [**Swift by Sundell**](https://www.swiftbysundell.com/) - Blog com artigos aprofundados e práticos sobre diversos aspectos do desenvolvimento Swift.
- [**Ray Wenderlich**](https://www.raywenderlich.com/) - Tutoriais de alta qualidade sobre desenvolvimento para plataformas Apple (Swift, SwiftUI, UIKit).
- [**AppCoda**](https://www.appcoda.com/) - Tutoriais e artigos sobre desenvolvimento iOS com Swift, do básico ao avançado.

## Frameworks Oficiais

### UI
- [**SwiftUI**](https://developer.apple.com/swiftui/) - Framework declarativo moderno para interfaces
- [**UIKit**](https://developer.apple.com/uikit/) - Framework tradicional para iOS
- [**AppKit**](https://developer.apple.com/appkit/) - Framework para macOS

### Dados e Persistência
- [**Core Data**](https://developer.apple.com/coredata/) - Persistência orientada a objetos
- [**SwiftData**](https://developer.apple.com/swiftdata/) - Persistência moderna (iOS 17+)

### Programação Assíncrona
- [**Combine**](https://developer.apple.com/combine/) - Programação reativa
- [**Async/Await**](https://docs.swift.org/swift-book/LanguageGuide/Concurrency.html) - Concorrência moderna

## Componentes e Bibliotecas

- [**Cocoa Controls**](https://cocoacontrols.com/) - Coleção de componentes de UI open source para iOS e macOS.
