# Frameworks de UI da Apple 

Este resumo apresenta os principais frameworks para construir interfaces de usuário nos sistemas operacionais da Apple, com destaque para o Apple Watch.

**1. SwiftUI:**
* **Plataformas:** **iOS**, **iPadOS**, **macOS**, **watchOS**, **tvOS**, visionOS.
* **Destaque para Apple Watch:** A **framework principal e recomendada** para construir interfaces de usuário em aplicativos para Apple Watch. Oferece uma abordagem declarativa e flexível para criar interfaces que se adaptam bem às telas do dispositivo.

**2. UIKit:**
* **Plataformas:** **iOS**, **iPadOS**, tvOS.
* **Destaque para Apple Watch:** **Não é usado diretamente** para construir a interface principal de aplicativos para Apple Watch. O SwiftUI é a tecnologia primária.

**3. AppKit:**
* **Plataforma:** **macOS**.
* **Destaque para Apple Watch:** **Não é relevante** para o desenvolvimento de interfaces para Apple Watch.
